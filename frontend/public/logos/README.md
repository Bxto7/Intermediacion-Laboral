# Logos de empresas

Coloca aquí los logos de las empresas que aparecen en la landing
("Empresas que confían en Linku").

## Cómo agregar un logo

1. Sube el archivo a esta carpeta `frontend/public/logos/`.
2. Nómbralo en minúsculas con guiones, igual que el `logo` definido en
   `frontend/src/pages/LandingPage.tsx` (sección COMPANIES). Ejemplos:
   - `cementos-andinos.svg`
   - `doe-run.png`
   - `caja-huancayo.webp`
3. Listo: aparecerá automáticamente. Si el archivo no existe, se muestra
   la inicial como respaldo (no se rompe nada).

## Formato recomendado

- **SVG** (ideal: nítido a cualquier tamaño) o **PNG/WebP**.
- **Fondo transparente** (los logos van sobre tarjeta clara).
- Tamaño aprox. **200×80 px** (o la proporción natural del logo).
- Peso: < 50 KB por logo.

> Asegúrate de tener permiso/derechos para usar cada logo.
