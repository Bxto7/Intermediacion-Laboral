# Logos institucionales / oficiales

Coloca aquí los logos del Estado y el logo de Linku.
Se usan en la barra superior (header, fondo NAVY oscuro) y en el footer (fondo NAVY oscuro).

## Archivos a subir (nómbralos EXACTAMENTE así)

| Archivo                                | Qué es                                   | Uso |
|----------------------------------------|------------------------------------------|-----|
| `gobierno-regional-junin.png`          | Gobierno Regional de Junín (color)       | Respaldo / fondos claros |
| `gobierno-regional-junin-blanco.png`   | Gobierno Regional de Junín (blanco/mono) | Barra navy + footer |
| `drtpe.png`                            | DRTPE (color)                            | Respaldo / fondos claros |
| `drtpe-blanco.png`                     | DRTPE (blanco/mono)                      | Barra navy + footer |
| `linku.png` (o `linku.svg`)            | Logo propio de Linku (color)             | Fondos claros |
| `linku-blanco.png` (o `.svg`)          | Logo propio de Linku (blanco/mono)       | Header + footer (navy) |

## Formato recomendado
- **PNG o SVG con fondo transparente.**
- Las versiones `*-blanco` deben ser monocromas claras (van sobre navy `#0d1b29`).
- Alto útil aprox. **40–48 px** (proporción natural). Peso < 80 KB.

> Si subes otro formato (jpg/webp), avísame y ajusto las rutas en el código.
